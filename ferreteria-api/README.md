# Ferreteria API

Sistema de gestión para ferretería con control de inventario y administración de pedidos.

## Requirements
- Java 17
- Maven
- MySQL

## Configuration
Ensure your MySQL database is running and update `src/main/resources/application.yml` with your credentials.

## Running the App
```bash
mvn spring-boot:run
```

## Swagger Documentation
Access the API documentation at: http://localhost:8080/swagger-ui.html
